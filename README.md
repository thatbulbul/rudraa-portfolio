# Rudru

A modern Next.js portfolio website built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Radix UI
- Lucide Icons

## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Project Structure

- `/app` - Next.js app directory
- `/components` - React components
- `/components/ui` - Reusable UI components
- `/hooks` - Custom React hooks
- `/lib` - Utility functions
- `/public` - Static assets
